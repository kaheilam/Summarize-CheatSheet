CSS - Display & Positioning
Position
Static Default value. No need to specify.
relative
  • Position element relative to its default static position on webpage.
  • Relative element will not affect the positioning of other elements
  • top, bottom, left, right
  • Can be specify in pixels, ems, %
absolute
  • All other elements on the page will ignore the absolute element.
  • Absolute element will be positioned relative to its closest positioned parent element.
  • top, bottom, left, right
  • Can be specify in pixels, ems, %
fixed
  • Fix the element to specific position on the page regardless of scrolling.
  • top, bottom, left, right
  • Can be specify in pixels, ems, %
sticky
  • Keep element flow as the user scroll, but sticks to a specific position.
  • top, bottom, left, right
  • Can be specify in pixels, ems, %
z-index Controls how far back / forward the element should appear on the webpage.
float
  • Moving element as far left / right
  • left, right
clear
  • Controls the flow next to float elements.
  • left, right, both, none
Display
inline
  • Element have a box that wraps tightly around their content
  • Only take up the amount of space necessary to display their content
  • Do not require a new line after each element

  • Cannot alter size by height or width properties
  • <a></a>, <em></em>, <strong></strong>
block
  • Element display in different line
  • Element fill the entire width of the page
  • <h1></h1>, <p></p>, <div></div>, <footer></footer>

CSS - Sizing Elements

  • Default base font-size set by the browser if none is given.
  • if base font-size = 16px, then 1em = 16px, 2em = 32px...

  • root em
  • Check root elements to size font.
  • root element is <html>

  • often used to size box-model values (width and height, padding, border, and margins).
